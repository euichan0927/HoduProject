# 😺 HODU 웹 페이지 서비스

## 프로젝트 소개

- HODU 라는 이름을 가진 고양이에 대한 정보가 담겨있는 웹 페이지 서비스입니다.
- HODU의 사진을 다운로드 하고 더 다양한 사진들을 볼 수 있습니다.
- HODU가 존재하는 위치를 지도로 확인할 수 있습니다.
- e-mail을 입력하여 구독서비스를 등록할 수 있습니다.

## 개발 환경

- HTML
- CSS
- JavaScript

→ Intellij
